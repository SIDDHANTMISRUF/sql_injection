# SQL Injection Lab - Web folder
- Start the project with `docker-compose up --build` from project root.
- Web app accessible at http://localhost:8080
- Pages:
  - /login_vuln.php  (vulnerable login)
  - /login_fix.php   (fixed login)
  - /search_vuln.php (vulnerable search)
  - /search_fix.php  (fixed search)
- Logs are written to query_logs.txt in this folder.
